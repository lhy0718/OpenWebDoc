# @openwebdoc/ui

Shared React UI components and styles for OpenWebDoc apps.

## Install

```sh
npm install @openwebdoc/ui
```

## Usage

```tsx
import { AppShell, ValidationPanel } from "@openwebdoc/ui";
import "@openwebdoc/ui/styles.css";
```

The stylesheet is included in the published package so apps can opt into the default OpenWebDoc UI theme.
