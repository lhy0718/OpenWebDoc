Safe package intake procedure

Runbook Version 0.1 Owner: document operations

Intake checklist

1. Receive Save the `.htmlx` package from a trusted channel.

2. Validate Run CLI validation before opening unknown content.

3. Inspect Confirm manifest resources are package-local.

4. Revise Use unpacked files for large changes.

5. Repack Validate the final file before distribution.

Escalation table

Issue Likely cause Response
Remote resource blocked Document references an external URL. Move the asset into the package or remove the reference.
Script blocked HTML contains executable JavaScript. Replace behavior with declarative HTML, CSS, or metadata.
Metadata warning LLM metadata is not marked as reference data. Add explicit user-visible assistant hints.
