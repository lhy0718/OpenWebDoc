Pitch deck

A document package that opens as software surface

Use this template for a product or startup story that needs crisp slides, local assets, and portable distribution.

Problem

Documents are still trapped between static files and web apps.

Static filesPortable but hard to revise with structure.
Web appsInteractive but difficult to archive as one document.
AI editsPowerful but need explicit file boundaries.

Solution

HTMLX packages keep HTML, CSS, assets, manifests, and metadata together.

Reader Opens a browser-native document.
Author Makes small corrections on the document surface.
Agent Edits unpacked package files and validates the result.

Ask

Test the package loop with one real document workflow.

Open, revise, validate, export, and compare whether the document remains readable without becoming a closed app.
