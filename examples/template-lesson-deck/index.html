Lesson deck

Teach a workflow as a browser-readable deck

A classroom or workshop template for concept, demonstration, practice, and recap.

Concept

A `.htmlx` file is a document package, not a script bundle.

InsideHTML, CSS, local assets, manifest, metadata.
OutsideProvider keys, hidden instructions, remote runtime code.

Practice

Learners inspect one package and mark what should stay local.

  1. Open the document.
  2. Find manifest resources.
  3. Validate the package.
  4. Explain the trust boundary.

Recap

Readable source, local resources, explicit validation.

Use the slide deck itself as an example of the format being taught.
